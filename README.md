## How do I use them?
### UCL officials
1. Take them away, make it responsive, apply them, don't need my consent.
2. Use a modern JavaScript framework to refactor the website (no jQuery pls)

### Chrome
1. Install [Stylus](https://chrome.google.com/webstore/detail/stylus/clngdbkpkpeebahjckkjfobafhncgmne) from Chrome web store
2. Copy code inside ```main.css```
3. Open Stylus menu, and click "Manage"
4. Click "Write new style"
5. Paste the code, give it a name, and then save.

### Other browsers
[Firefox add-ons](https://addons.mozilla.org/firefox/addon/styl-us/)

[Opera add-ons](https://addons.opera.com/extensions/details/stylus/) (see [wiki](https://github.com/openstyles/stylus/wiki/Opera,-Outdated-Stylus) for more recent version)
